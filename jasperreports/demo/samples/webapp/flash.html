  Flash Viewer

Once generated on the server side, as a result of the report filling process, the report can be viewed on the client side using a Flash viewer.

This viewer is a Flash movie component built in a separate project available at JasperReports Flash Viewer, and comes in the form of a SWF file called jasperreports-flash-x.x.x.swf.

But first, you have to make sure your browser supports Flash.

You can play Flash movies in any browser that has the appropriate version of the Adobe Flash Player Plug-in installed.

If your browser can play Flash movies, you should be able to see a clock displaying the current time below:



If your browser passed the above test and you are able to play Flash movies with it, then you are ready to test the JasperReports Flash Viewer and see the sample report.
